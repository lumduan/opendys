# Code of Conduct

opendys adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
(v2.1). In short: be kind, welcoming, patient, and respectful. Harassment, discrimination, and
demeaning behavior are not tolerated.

This project exists to help people — especially children — with dyslexia read more easily. We ask
contributors to hold to that spirit in every interaction.

Report unacceptable behavior by opening a confidential issue or contacting the maintainers through
GitHub. Maintainers will review and respond in good faith.
